# 📡 WSN Analytics Dashboard

An Automated Performance Analysis Tool for **Wireless Sensor Networks (WSN)** built using Python, Streamlit, SQL, and Data Visualization techniques.

---

## 🚀 Features

- 📂 Upload sensor network CSV data
- 📊 Interactive Streamlit dashboard
- 📈 Real-time performance metrics visualization
- 📉 Advanced analytics and statistical analysis
- 🗄️ SQLite database integration
- 📄 Automated PDF report generation
- 📡 REST API support using Flask
- 🎨 Interactive Plotly visualizations

---

## 📊 Key Metrics

The dashboard analyzes:

- Packet Delivery Ratio (PDR)
- Network Latency
- Energy Consumption
- Network Lifetime
- Node Health Score

---

## 📈 Visualizations Included

- Latency Distribution Histogram
- PDR per Node Analysis
- Energy Consumption Graph
- Energy vs Lifetime Bubble Chart
- Correlation Heatmap
- Radar Analysis
- Lifetime Trend Analysis

## ▶️ Run the Project

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Streamlit Dashboard

```bash
streamlit run dashboard_app.py
```

### Run Main Analysis

```bash
python main.py
```
